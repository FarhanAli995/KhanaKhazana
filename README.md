## 📌 **Project Overview**

**Khanakhazana** is a modern, full-stack restaurant web application showcasing traditional cuisine through an elegant digital experience. Built with **Django** powering the backend and **Tailwind CSS** crafting the frontend, this project demonstrates clean architecture and responsive design principles.

The platform offers users an immersive browsing experience to explore authentic dishes, view detailed culinary presentations, and interact with a polished food ordering interface.

> 🎯 **Purpose:** Hands-on practice in full-stack development using Django ecosystem and modern CSS frameworks.

## 🚀 **Quick Start Guide**

### **1️⃣ Clone Repository**
```bash
git clone https://github.com/yourusername/khanakhazana.git
```

### **2️⃣ Setup Environment**
```bash
cd khanakhazana
python -m venv venv

# Windows
venv\Scripts\activate

# macOS/Linux
source venv/bin/activate
```

### **3️⃣ Install Dependencies**
```bash
pip install django
npm install
```

### **4️⃣ Launch Server**
```bash
python manage.py runserver
```
🔓 **Access:** `http://127.0.0.1:8000/`
